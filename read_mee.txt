-Mon projet :

La base DVF enregistre les données des transactions immobilières de tous les communes en frances métropoles (hors Alsace et Moselle) ainsi que les département d'Outre-Mer (hors Mayotte). 
À partir de ce jeux de données, nous pourons faire des analyses pour la valeur foncières en france selon différentes critères 
 Nous pourrons savoir aussi à partir de ce jeu de données l'évoution selon le temps ( entre 2016 et 2020 ) pour chaque type de biens. Le point faible de ce jeu, c'est qu'il n'y a pas mal des informations manquantes (comme les caractéristiques du bien, le nombre d'étage, ascenseur, année de construction ...)

1) j'ai commencé par le traitement des données 

2) clean les dataset avec le changement des types ( jeu de données contient aussi beaucoup des données manquants. 

3) faire des vizualizations pour arriver à des résultats

4) j'ai utilisé le site de l'éleve et j'ai pris environ 350 000 ligne pour pouvoir l'utilisé sur streamlit.
